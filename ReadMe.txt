How to launch the website in django

0. download the project
1. go to the directory .\Main in the command line
2. write    python manage.py runserver
3. go to http://127.0.0.1:8000/ecosyn/ to view the site
4. or to http://127.0.0.1:8000/admin/ to administrate it
